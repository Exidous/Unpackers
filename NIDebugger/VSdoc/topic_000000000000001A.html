NIDebugger.FindProcAddress Method

See Also

Finds the address for the given method inside the given module. The method requested must be exported to be found. This is equivalent to the GetProcAddress() Win32 API but takes into account ASLR by reading the export tables directly from the loaded modules within the debugged process.
